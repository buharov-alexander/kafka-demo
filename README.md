# kafka-demo
It is the demo project to work with Apache Kafka

## Launch Kafka
You can use docker-compose to launch Kafka + Zookeeper
```
docker-compose up
```
## Launch Demo Project
Start Spring-boot application
```
cd app
mvn spring-boot:run
```
## Check demo
Application returns UI page which generated by Vaadin
```
http://localhost:8080/
```
